# 📋 Чекер статьи — Вестник ЕНУ

Автоматическая проверка соответствия статьи требованиям журнала
«Вестник ЕНУ им. Л.Н. Гумилева» (серия Химия/География, шаблон 2025).

## Запуск локально

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Деплой на Streamlit Cloud (бесплатно)

1. Создайте репозиторий на GitHub и добавьте `app.py` и `requirements.txt`
2. Перейдите на https://share.streamlit.io
3. Нажмите **New app** → выберите репозиторий → **Deploy**
4. Приложение будет доступно по публичной ссылке

## Что проверяется

| № | Критерий |
|---|----------|
| 1 | Объём статьи ≥ 3500 слов |
| 2 | Аннотация рус. ≤ 300 слов |
| 3 | Аннотация каз. (аңдатпа) |
| 4 | Abstract (англ.) |
| 5 | Ключевые слова 3–10, разделитель «;» |
| 6 | Код МРНТИ |
| 7 | ORCID авторов |
| 8-12 | Разделы: Введение, М&М, Результаты, Обсуждение, Заключение |
| 13 | Вклад авторов (CRediT) |
| 14 | Финансирование |
| 15 | Конфликт интересов |
| 16 | ≥ 25 источников |
| 17 | DOI в ссылках |
| 18 | Стиль цитирования APA-7 |
| 19 | Вспомогательный материал |
