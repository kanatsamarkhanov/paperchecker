## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install streamlit python-docx pillow pandas openpyxl
```

### 2. Запуск

```bash
streamlit run app.py
```

### 3. Использование

1. Откройте браузер по адресу `http://localhost:8501`
2. Загрузите `.docx`-файл статьи через кнопку загрузки
3. Получите результаты проверки по 28 критериям

## 📊 Критерии проверки (28 пунктов)

| № | Критерий | Требование |
|---|----------|------------|
| 0 | Название статьи | Строки 3–4 документа |
| 1 | Язык статьи | Определяется автоматически |
| 2 | Объём | ≥ 3 500 слов |
| 3 | Аннотация (рус) | ≤ 300 слов |
| 4 | Аннотация (каз) | Обязательно |
| 5 | Abstract (англ) | Обязательно |
| 6 | Ключевые слова | 3–10, разделитель «;» |
| 7 | МРНТИ / IRSTI | Обязательно |
| 8 | ORCID авторов | Для каждого автора |
| 9 | §1. Введение | Обязательно |
| 10 | §2. Материалы и методы | Обязательно |
| 11 | §3. Результаты | Обязательно |
| 12 | §4. Обсуждение | Обязательно |
| 13 | §5. Заключение | Обязательно |
| 14 | §6. Вспомогательный материал | Рекомендуется |
| 15 | §7. Вклад авторов (CRediT) | Обязательно при > 1 авторе |
| 16 | §8. Информация об авторе | Рекомендуется |
| 17 | §9. Финансирование | Обязательно |
| 18 | §10. Благодарности | Рекомендуется |
| 19 | §11. Конфликт интересов | Обязательно |
| 20 | Список литературы | ≥ 10 источников |
| 21 | Стиль ссылок | APA / Vancouver / ГОСТ + DOI |
| 22 | Формат бумаги | A4 (210 × 297 мм) |
| 23 | Поля | Все поля 20 мм |
| 24 | Шрифт | Times New Roman, 12 pt |
| 25 | Таблицы | Должны присутствовать |
| 26 | Рисунки | Мин. 300 DPI |
| 27 | Многоязычные аннотации | 2 аннотации на других языках |

## 🔍 Проверка стиля списка литературы

Функция `detect_refs_style()` автоматически определяет стиль оформления:

| Стиль | Признак обнаружения |
|-------|---------------------|
| **APA** | Год в скобках после авторов: `Иванов, И.И. (2023).` |
| **Vancouver** | Номер в начале строки: `[1]` или `1.` |
| **ГОСТ** | Фамилия с инициалами: `Иванов И.И.` |

Дополнительно подсчитывается количество **DOI-ссылок** в тексте ссылок.

## 🖼️ Анализ рисунков

Для каждого встроенного изображения отображается:
- Размер в пикселях
- Физический размер в документе (мм)
- Расчётный DPI
- Встроенный DPI (из метаданных)
- Формат файла (PNG, JPEG, …)
- Статус: ✅ ≥ 300 DPI / ❌ < 300 DPI / ⚠️ нет данных
