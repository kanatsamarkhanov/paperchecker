# 📋 Мақаланың дайындығын тексеруші / Проверка готовности статьи — Вестник ЕНУ

🇰🇿 Л.Н. Гумилев атындағы ЕҰУ Хабаршысы (Химия/География сериясы, 2025 үлгісі) талаптарына мақаланың сәйкестігін автоматты түрде тексеруге арналған веб-қосымша.

🇷🇺 Автоматическая проверка соответствия статьи требованиям журнала «Вестник ЕНУ им. Л.Н. Гумилева» (серия Химия/География, шаблон 2025).

---

## 🌟 Мүмкіндіктері / Особенности

- **Қос тілді интерфейс / Двуязычный интерфейс:** Қазақша және Русский
- **Тақырыптар / Темы:** Күндізгі және Түнгі режимдер (Light / Dark mode)
- **Кешенді тексеру / Комплексная проверка:** Мәтіндік құрылымнан бастап, қағаз форматы мен шрифттерге дейін тексеру

---

## 🚀 Жергілікті іске қосу / Запуск локально

```bash
pip install -r requirements.txt
streamlit run app.py
```

---

## ☁️ Streamlit Cloud-қа орналастыру / Деплой на Streamlit Cloud (тегін / бесплатно)

1. GitHub-та репозиторий жасап, `app.py` және `requirements.txt` файлдарын жүктеңіз. / Создайте репозиторий на GitHub и добавьте файлы.
2. [share.streamlit.io](https://share.streamlit.io) сайтына өтіңіз. / Перейдите на сайт.
3. **New app** батырмасын басып → репозиторийді таңдаңыз → **Deploy** басыңыз.
4. Қосымша жалпыға қолжетімді сілтеме бойынша ашылады. / Приложение будет доступно по публичной ссылке.

## 🔍 Не тексеріледі? / Что проверяется?

| № | Критерий (Қазақша) | Критерий (Русский) |
|---|---|---|
| 1 | Мақала көлемі ≥ 3500 сөз | Объём статьи ≥ 3500 слов |
| 2 | Орысша аңдатпа ≤ 300 сөз | Аннотация рус. ≤ 300 слов |
| 3 | Қазақша аңдатпаның болуы | Аннотация каз. (аңдатпа) |
| 4 | Ағылшынша Abstract болуы | Abstract (англ.) |
| 5 | Түйінді сөздер (3–10, бөлгіш «;») | Ключевые слова (3–10, разделитель «;») |
| 6 | МРНТИ коды | Код МРНТИ |
| 7 | Авторлардың ORCID идентификаторы | ORCID авторов |
| 8–12 | Бөлімдер: Кіріспе, М&Ә, Нәтижелер, Талқылау, Қорытынды | Разделы: Введение, М&М, Результаты, Обсуждение, Заключение |
| 13 | Авторлар үлесі (CRediT форматы) | Вклад авторов (CRediT) |
| 14 | Қаржыландыру туралы ақпарат | Финансирование |
| 15 | Мүдделер қақтығысы | Конфликт интересов |
| 16 | Әдебиеттер саны ≥ 25 | ≥ 25 источников (research paper) |
| 17 | Сілтемелердегі DOI көрсеткіштері | DOI в ссылках |
| 18 | APA-7 дәйексөз келтіру стилі | Стиль цитирования APA-7 |
| 19 | Қағаз форматы (A4: 210×297 мм) | Формат бумаги (A4: 210×297 мм) |
| 20 | Жақтаулар (~2 см) | Поля (~2 см) |
| 21 | Шрифт және кегль (Times New Roman, 11/12) | Шрифт и кегль (Times New Roman, 11/12) |
| 22 | Кестелердің болуы | Наличие таблиц |
| 23 | Суреттер (мин. 300 DPI) | Рисунки (мин. 300 DPI) |
