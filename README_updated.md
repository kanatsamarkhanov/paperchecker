# 📋 Мақаланы тексеруші / Чекер статьи — Вестник ЕНУ

🇰🇿 Л.Н. Гумилев атындағы ЕҰУ Хабаршысы (Химия/География сериясы, 2025 үлгісі) талаптарына мақаланың негізгі құрылымдық элементтерінің сәйкестігін автоматты түрде тексеруге арналған веб‑қосымша.
🇷🇺 Веб‑приложение для автоматической проверки основных структурных требований журнала «Вестник ЕНУ им. Л.Н. Гумилева» (серия Химия/География, шаблон 2025).

---

## 🌟 Мүмкіндіктері / Особенности

- Қос тілді интерфейс / Двуязычный интерфейс: **Қазақша** және **Русский**.
- Тақырыптар / Темы: **Күндізгі** және **Түнгі** режимдер (Light / Dark mode).
- Кешенді тексеру / Комплексная проверка:
  - автор туралы ақпарат, тіл, көлем;
  - үш тілдегі аңдатпалар / аннотации;
  - негізгі ғылыми бөлімдер (Кіріспе, Материалдар мен әдістер, Нәтижелер, Талқылау, Қорытынды);
  - авторлардың үлесі (CRediT), қаржыландыру, алғыстар, мүдделер қақтығысы;
  - қағаз форматы, жақтаулар, шрифт, кестелер мен суреттер.

> Назар аударыңыз: қазіргі нұсқада **әдебиеттер тізімінің саны, DOI және APA‑7 стилі автоматты түрде тексерілмейді** – оларды редактор қолмен қарайды.

## 🚀 Жергілікті іске қосу / Запуск локально

```bash
pip install -r requirements.txt
streamlit run app.py
```

Бұдан кейін браузерде әдетте `http://localhost:8501` адресі автоматты түрде ашылады.

## 🔍 Не тексеріледі? / Что проверяется?

| №  | Критерий (Қазақша)                                  | Критерий (Русский)                                      |
|----|-----------------------------------------------------|---------------------------------------------------------|
| 1  | Мақала көлемі ≥ 3500 сөз                            | Объём статьи ≥ 3500 слов                                |

| 3  | Қазақша аңдатпаның болуы                            | Наличие аннотации каз.                                  |
| 4  | Ағылшынша Abstract болуы                            | Наличие Abstract (англ.)                                |
